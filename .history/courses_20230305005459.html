Conheça nossos cursos

Desenvolvedor Web

Neste curso, você aprenderá as principais tecnologias utilizadas no desenvolvimento de sites, tais como HTML, CSS e JavaScript, além de frameworks como React e Node.js. Com este curso, você estará preparado para criar sites modernos e responsivos, e terá uma base sólida para se aprofundar em outras tecnologias.

Desenvolvedor Mobile

Este curso é voltado para quem deseja desenvolver aplicativos para smartphones e tablets. Você aprenderá a programar em linguagens como Java e Kotlin para Android e Swift para iOS, além de utilizar frameworks como React Native para criar aplicativos multiplataforma.

Back-End

O desenvolvimento de sistemas é uma parte fundamental de qualquer projeto de software. Com este curso, você aprenderá as habilidades necessárias para criar aplicativos que se comunicam com bancos de dados, realizam autenticação de usuários e gerenciam recursos.

Dados com Python

Com a crescente quantidade de dados gerados diariamente, a análise de dados se tornou uma habilidade essencial em muitas áreas. Com este curso, você aprenderá a trabalhar com dados em Python, desde a coleta e limpeza até a visualização e modelagem.

UX/UI Designer

O design é uma parte fundamental do desenvolvimento de aplicativos e sites. Com este curso, você aprenderá a criar interfaces atraentes e funcionais, que melhorem a experiência do usuário e aumentem a interação com seus clientes.

Jogos com C#

O desenvolvimento de jogos é um mercado em constante crescimento, e com este curso você poderá aprender a criar jogos em uma das linguagens mais utilizadas na indústria. Desde a criação de gráficos até a implementação de lógica de jogo, este curso oferece uma abordagem abrangente para a criação de jogos.
